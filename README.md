# 🌍 AI Map Generator

### Turn natural language into interactive maps

Build maps from plain text.

> “Shows maps based on AI queries”
> → instantly get structured locations, filters, and an interactive map.

---

## 🚀 Live Demo

👉 https://storelocatormap.komliev.studio/

---

## 🎯 What is this?

AI Map Generator is a **map-first SaaS application** that converts user queries into:

* 📍 Realistic locations
* 🗺️ Map visualization
* ⚙️ Dynamic filters
* 🌦 Contextual data (weather)

All powered by AI + structured JSON.

---

## ✨ Why it matters

Most map apps require:

* manual search
* rigid filters
* predefined categories

This app replaces that with:

👉 **Natural language → dynamic map UI**

---

## 🧠 How it works

```txt
User query
↓
AI (structured JSON)
↓
Backend validation
↓
Frontend rendering (Leaflet)
↓
Interactive map
```
